# landing-page-yasmin

Landing page pessoal e portfólio digital desenvolvida para **Yasmin Eustáquio**, estudante de Artes Visuais na UFPB, com o objetivo de apoiar sua busca por oportunidades profissionais na área criativa e cultural.

## Sobre o projeto

Este projeto é uma página estática em HTML/CSS/JS que apresenta o perfil artístico e acadêmico de Yasmin de forma visual e profissional. O design foi inspirado na Casa **Lufa-Lufa** de Harry Potter — referência pessoal da artista — traduzindo valores como dedicação, lealdade e trabalho em uma identidade visual com paleta dourada e atmosfera aconchegante.

## Funcionalidades

- Navegação fixa com indicador de seção ativa
- Hero animado com arco gótico e partículas de vela
- Seção **Sobre** com biografia e vínculo com a OSUFPB
- Seção **Portfólio** com grid de obras e espaços para trabalhos futuros
- Seção **Texto Autoral** com produção literária própria
- Seção temática da **Casa Lufa-Lufa**
- Seção de **Contato** com link para o Instagram
- Totalmente responsivo para mobile e desktop
- Arquivo único `.html` — sem dependências externas além de fontes Google

## Tecnologias

- HTML5 semântico
- CSS3 (variáveis, Grid, Flexbox, animações)
- JavaScript vanilla (Intersection Observer)
- Google Fonts: Cinzel + Crimson Pro

## Como usar

Basta abrir o arquivo `yasmin_eustaquio.html` diretamente no navegador — nenhuma instalação necessária.

```bash
# Ou servir localmente
npx serve .
```
